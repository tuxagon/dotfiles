# Fresh setup

1. Install homebrew and all associated formulae.

```bash
$ ./setup/homebrew.sh
```

2. Install and set up neovim.

```bash
$ ./setup/neovim.sh
```
_For iTerm2, ensure the Option key works correctly for moving between panes after setup_

3. Install oh-my-zsh

```bash
$ ./setup/zsh.sh
```
